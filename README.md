# ⚠ Códigos de falla registrados

## JBBF

**A6CF**

Se refiere a un problema con el sensor AUC (Control Automático de Recirculación de Aire). Pendiente determinar su influencia en la falla principal.

---

## EGS

**578E**

El fluido de la transmisión se ha degradado o ha alcanzado su límite de desgaste preprogramado. Pendiente evaluar su relación con el apagado del motor.

---

## DSC

**5DE0**

Error de plausibilidad del sensor de desgaste de las pastillas de freno delanteras o error de inicialización incorrecta/no inicializada de las pastillas de freno delanteras. Actualmente no se considera la causa principal.

---

## FRM

**A8B6**

Mal funcionamiento de las luces de circulación diurna (DRL) o de las luces de freno.

**A8AE**

Fallo en el circuito del faro antiniebla izquierdo.

**A8AF**

Problema relacionado con el faro del lado derecho.

**9CB6**

Bobina de control de alineación vertical del faro.
Actualmente se consideran fallas secundarias hasta obtener mayor evidencia.

---

## Red MOST

El módulo del cargador de CD ubicado en el maletero no responde.

Actualmente no existe evidencia de que influya directamente en la falla principal del motor.

# 🤖 Filosofía del proyecto

Este repositorio busca documentar el diagnóstico y reparación del vehículo de forma estructurada.

Toda conclusión debe estar respaldada por pruebas o evidencia objetiva.

La inteligencia artificial se utiliza como herramienta de apoyo para organizar información, generar hipótesis y sugerir próximos pasos, pero las decisiones finales deben basarse en datos obtenidos durante el diagnóstico.
